### 2. ATTACK VECTORS & SCENARIOS

**Vector 1: Cross-Site Scripting (XSS)**
- **Entry Point**: js/repos.js, js/orgs.js rendering user-controlled data
- **Attack Scenario**: Attacker creates GitHub repository with malicious name containing `<script>` tags. When application renders repository list without sanitization, script executes in victim's browser.
- **Impact**: Session hijacking, credential theft, malware distribution
- **Likelihood**: HIGH
- **Mitigation**: Input validation, output encoding, CSP headers

**Vector 2: Supply Chain Attack**
- **Entry Point**: package.json dependencies
- **Attack Scenario**: Attacker compromises npm package used by application or publishes typosquatted package. Malicious code executes during build or runtime.
- **Impact**: Complete system compromise, data exfiltration, backdoor installation
- **Likelihood**: MEDIUM
- **Mitigation**: Lock dependencies, use npm audit, implement SRI for CDN resources

**Vector 3: Information Disclosure**
- **Entry Point**: Console.log statements, error messages
- **Attack Scenario**: Attacker opens browser developer tools while using application, captures logged API responses containing tokens, user data, or internal system information.
- **Impact**: Credential exposure, API abuse, reconnaissance for further attacks
- **Likelihood**: HIGH
- **Mitigation**: Remove console statements, implement proper logging, sanitize error messages

**Vector 4: Server Misconfiguration**
- **Entry Point**: .htaccess configuration
- **Attack Scenario**: Misconfigured access rules allow directory traversal, expose .git folder, or permit unauthorized file access.
- **Impact**: Source code exposure, configuration leak, full system compromise
- **Likelihood**: MEDIUM
- **Mitigation**: Security audit of server configuration, principle of least privilege

**Vector 5: API Abuse**
- **Entry Point**: Unprotected API calls in JavaScript
- **Attack Scenario**: Attacker reverse-engineers API calls, extracts any embedded credentials, and abuses API without rate limiting.
- **Impact**: Service disruption, cost inflation, data scraping
- **Likelihood**: HIGH
- **Mitigation**: Server-side API proxy, rate limiting, token rotation
