## Setup Instructions

1. Add EC2_HOST secret with your EC2 instance's public IP address
2. Add EC2_USERNAME secret (e.g., 'ubuntu' or 'ec2-user' depending on your AMI)
3. Add EC2_SSH_PRIVATE_KEY secret with your EC2 key pair's private key content
4. Ensure your EC2 security group allows inbound traffic on port 80 (HTTP) and port 22 (SSH)
5. Install Docker on your EC2 instance if not already installed: sudo yum install docker -y && sudo systemctl start docker
6. Review and merge the PR to enable automatic deployments on push to master
