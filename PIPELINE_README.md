## Setup Instructions

1. Ensure your EC2 instance has the SSM Agent installed and running
2. Attach an IAM role to your EC2 instance with AmazonSSMManagedInstanceCore policy
3. Create an IAM user with ssm:SendCommand and ssm:GetCommandInvocation permissions
4. Add the required secrets to your GitHub repository (Settings → Secrets and variables → Actions)
5. Merge the PR and push to master branch to trigger the deployment

## How to Add Secrets

3. Click **New repository secret**
4. Add each secret listed above
